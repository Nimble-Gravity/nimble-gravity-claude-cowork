# Default memo format

Used when the user doesn't specify a format. This is the generic template — a client
engagement swaps in the client's letterhead, section names, and house style here (this
file is the client-swappable seam for the skill).

```
MEMO

To:       [audience]
From:     [author]
Date:     [date]
Re:       [one-line subject]

PURPOSE
  One or two sentences. State the bottom line — the conclusion or the ask — first.

BACKGROUND
  The minimum context a reader needs. Cite source files/notes for any fact.

KEY POINTS
  - Point, with its source.
  - Point, with its source.
  - [NEEDS INPUT: …] for anything the source material doesn't support.

RECOMMENDATION / NEXT STEPS
  What should happen next, who owns it, and any open items flagged for review.
```

## Variants

- **Summary memo** — drop Recommendation; end with Key Points.
- **Status memo** — replace Background with "Since last update".
- **Recommendation memo** — lead Purpose with the recommendation verbatim.
